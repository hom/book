# 套接字编程简介

IPv4套接字结构

```c
// netinet/in.h
struct in_addr {
  in_addr_t s_addr; // 32位IPv4地址
}

struct sockaddr_in {
  // 套接字长度大小
  unint8_t sin_len;
  // AF_INET
  sa_family_t sin_family;
  // 16位TCP或者UDP端口
  in_port_t sin_port;
  // 32位IPv4地址
  struct in_addr sin_addr;
  // 暂未使用
  char sin_zero[8];
}
```
